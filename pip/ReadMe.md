

# 安装pip 

## 快速安装pip

```
wget https://raw.githubusercontent.com/caozhilong/easypack/master/pip/pip_mgnt.sh

chmod 777 pip_mgnt.sh

./pip_mgnt.sh install pip


# 可选动作和模块

Usage: ./pip_mgnt.sh ACTION MODULE
       ACTION: install|uninstall|show|check
       MODULE: module name, eg: pip|tensorflow|numpy|matplotlib

```
